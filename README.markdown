Wallet
===

See what's inside of your ender $
-------

__To install:__

    npm install ender-wallet

Easy.

__To run:__

cd into a directory with an ender build, then run...

    ender-wallet open

Then hit localhost 8083 (in webkit||firefox) and you'll see what inside's your ender wallet.
